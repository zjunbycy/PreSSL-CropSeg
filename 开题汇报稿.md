# 开题汇报：基于预训练自监督学习模型的时序遥感耕地分割

> **课题：** ZJU-GISLAB-COURSE-2026 课程项目（第一题）
> **周期：** 3周
> **数据集：** PASTIS（ICCV 2021）
> **预训练模型：** Galileo（ICML 2025）

---

## 一、问题

**单张卫星图区分不了作物。** 同一时刻，不同作物可能长得一样——5月的小麦和5月的大麦都是绿色。但看完整年：小麦"绿→黄→收割"，大麦"裸土→绿→黄→收割"。每条作物有独特的物候指纹。

**所以需要时序。** Sentinel-2 提供 38-61 期多光谱影像，每个像素有一条时间曲线，靠这条曲线区分作物类型。

**但标注极贵。** PASTIS 只有 2433 个标注样本，不够从头训练深度模型。于是用自监督预训练——Galileo 在海量无标注卫星数据上学会了遥感场景的通用特征，我们只需要迁移到耕地分割任务。

**核心问题：Galileo 的自监督预训练在耕地分割上比 ImageNet 预训练好多少？**

---

## 二、技术方案

### 架构

```
Input: [B, T, 10, 128, 128]    Sentinel-2 时序
        │
Module 1: Galileo Encoder       自监督预训练的遥感视觉底座
        逐帧编码 → 4 个空间尺度特征 [F1~F4]
        每个 F_i: [B, T, C_i, H_i, W_i]
        │
Module 2: Temporal Fusion       时序注意力融合
        1×1 Conv → Softmax → 加权求和
        T 维坍缩 → 纯空间特征 [S1~S4]
        │
Module 3: DPT Decoder           Transformer 风格分割头
        Reassemble → Fusion → 上采样
Output: [B, 19, 128, 128]       作物分割图
```

### 为什么选 Galileo 而不是其他

| | ImageNet 预训练 | Galileo SSL 预训练 |
|---|---|---|
| 预训练数据 | 128 万张猫狗自然图 | 12.7 万全球卫星+天气+高程 |
| 光谱波段 | 3 个（RGB） | 10 个（Sentinel-2 全波段） |
| 时序理解 | 无 | 经过时空 mask 训练，理解物候规律 |
| 对耕地任务 | 相当于一个"懂拍照的人" | 相当于一个"懂遥感的人" |

## 三、实验（只做一个核心消融）

**只回答一个问题：Galileo 预训练比 ImageNet 预训练强在哪？**

| 组别 | Encoder 初始化 | 策略 | 说明 |
|------|---------------|------|------|
| ① | **随机初始化** | 全模型微调 | 下界——没有预训练 |
| ② | **ImageNet 预训练** | 线性探测 | 冻结 encoder，只训 fusion+decoder |
| ③ | **ImageNet 预训练** | 微调 | 解冻，小 lr 全模型训练 |
| ④ | **Galileo 预训练** | 线性探测 | 冻结 encoder，只训 fusion+decoder |
| ⑤ | **Galileo 预训练** | 微调 | 解冻，小 lr 全模型训练 |

**这张 5 行对比表就是你论文里最核心的结果。** ④ vs ② 证明 Galileo 的特征质量，⑤ vs ③ 证明端到端迁移效果，③ vs ① 证明预训练本身的价值。
